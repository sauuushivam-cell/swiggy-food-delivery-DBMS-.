# Swiggy Food Delivery Database Management System

## Project Overview
This project is a DBMS-based Swiggy Food Delivery Database created using SQL. It manages customers, restaurants, menu items, orders, delivery partners, payments, and reviews.

## Features
- Customer Management
- Restaurant Management
- Menu Management
- Order Management
- Delivery Partner Management
- Payment Management
- Review Management

## Technologies Used
- SQL
- DBMS
- Relational Database

## Database Tables
- Customers
- Restaurants
- Menu_Items
- Orders
- Order_Items
- Delivery_Partners
- Payments
- Reviews
